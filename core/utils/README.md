# 🛠️ Utilities Layer

This folder contains shared helper functions and data processing scripts used across the application.

### 📄 Files
- **`data_loader.py`**: Functions for reading CSV data and preprocessing it into a format suitable for the ML models.

### ⚙️ How it works
It abstracts the data loading logic away from the main application flow, ensuring that data is cleaned and prepared (e.g., converting genre strings to lists) before it is used by the system.
